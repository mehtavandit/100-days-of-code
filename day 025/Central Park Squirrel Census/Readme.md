## main.py
- The number of squirrels is counted based on the color of their fur.
- The information gathered is saved in a dictionary, which is then converted to a CSV file using the pandas library.

## squirrel data.csv
- This file contains the information about 3024 squirrels in Central Park.

## squirrel_count.csv
- This file contains data on the number of squirrels present based on their fur color.
